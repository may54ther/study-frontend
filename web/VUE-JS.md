## Computed vs Methods

> - computed
>   - 이전에 계산된 값을 캐시해두었다가 함수 호출시 재사용
>   - 데이터가 자주 변동되지 않을 때 사용
> - methods
>   - 사용될 때마다 함수의 계산을 다시 함
>   - 데이터가 수시로 업데이트 될 때 사용
